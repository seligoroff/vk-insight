# VK Insight - Консольный инструмент для работы с VK API

![PHP](https://img.shields.io/badge/PHP-8.1%2B-777BB4?style=flat-square&logo=php)
![Laravel](https://img.shields.io/badge/Laravel-9.x-FF2D20?style=flat-square&logo=laravel)
![License](https://img.shields.io/badge/License-MIT-green?style=flat-square)

Консольное приложение на Laravel для автоматизации работы с VKontakte API. Инструмент предоставляет набор команд для анализа постов, комментариев, поиска контента и мониторинга активности в группах VK.

## 🚀 Возможности

- **Поиск контента для републикации** - автоматический поиск постов без репостов
- **Поиск по слову** - поиск постов, содержащих указанное слово, с выводом статистики
- **Поиск дублей** - нахождение повторяющихся постов по текстовому паттерну
- **Анализ комментариев** - поиск комментариев конкретного автора во всех постах группы
- **Мониторинг групп** - проверка последних постов в списке групп с метриками (лайки, репосты, подписчики)
- **Работа с постами** - получение постов, комментариев, закрепление постов
- **Аналитика эффективности** - углубленный анализ постов с расчетом ER, лучшего времени публикации и сравнением периодов
- **Работа с фотоальбомами** - получение списка альбомов с фильтрацией и экспортом
- **Работа с аудио** - добавление аудио в группы
- **Проверка токена** - проверка прав доступа токена VK API

## 📋 Требования

- PHP 8.1+ (рекомендуется PHP 8.2+)
- Composer
- MySQL 5.7+ или MariaDB 10.3+ (рекомендуется MySQL 8.0+)
- Токен доступа VK API

## ⚡ Быстрый старт

```bash
# Клонируйте репозиторий
git clone https://github.com/seligoroff/vk-insight.git
cd vk-insight

# Установите зависимости и настройте проект
make setup

# Настройте токен VK API в .env
# VK_TOKEN=ваш_токен_vk_api

# Готово! Используйте команды
php artisan vk:check
```

## 📖 Основные команды

```bash
# Проверка последних постов в группах
php artisan vk:check

# Поиск постов по слову
php artisan vk:word концерт --owner=-12345678 --from=2024-01-01 --db

# Проверка прав доступа токена
php artisan vk:token-check

# Получение информации о группах
php artisan vk:groups-info

# Получение постов за период
php artisan vk:posts-get --owner=-12345678 --from=2024-01-01

# Получение постов для всех групп из vk-groups.csv
php artisan vk:posts-get-all --from="last month"

# Получение списка фотоальбомов
php artisan vk:albums-get --owner=-12345678

# Анализ эффективности постов
php artisan vk:analytics --owner=-12345678 --period=month --best-time

# Поиск кандидатов для републикации
php artisan vk:candidate --owner=-12345678 --noreposts
```

Полный список команд и их описание смотрите в [документации по командам](docs/commands.md).
